# Pro Number Guessing Game (Java)

An interactive console-based number guessing game built in Java. The game features multiple difficulty levels, limited attempts, and a high score tracking system.

## Features

* Multiple difficulty levels (Easy, Medium, Hard)
* Random number generation
* Limited number of attempts
* High score tracking
* Replay option
* Input validation with fallback defaults

## Technologies Used
* Java
* Random class
* Scanner (User Input Handling)
* Control Structures (Loops & Switch)
* Static Variables for High Score Tracking

## How to Play

1. Choose a difficulty level:

   * Easy (1–50)
   * Medium (1–100)
   * Hard (1–200)
2. You get 7 attempts to guess the number.
3. The system gives hints:

   * ⬆ Too Low
   * ⬇ Too High
4. Beat your best score to set a new high score!


## ▶ How to Run

### Compile:

bash
javac NumberGuessingGame.java

### Run:

bash
java NumberGuessingGame
